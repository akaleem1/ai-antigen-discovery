# AI Antigen Discovery

Exploring machine learning approaches to prioritize vaccine antigen candidates from pathogen proteomes.

## Motivation
Modern vaccine discovery increasingly relies on computational methods to identify promising antigen targets. This project explores how protein language models and machine learning can help prioritize antigen candidates from pathogen protein datasets.

## Approach
• Generate protein embeddings using pretrained protein language models  
• Analyze embedding space with dimensionality reduction and clustering  
• Train predictive models to rank antigen candidates  
• Explore integration of immunological features and structural annotations
